# Discord-Tool

## Features

- Server Nuke
  - Members
  - Roles
  - Emoji
  - Stickers
  - Channels

- Discord Account Nuke
  - Delete Owned Server
  - Leave all Server
  - Change Owned Server name
  - Transfer Owned Server to your Account

### Setup
1. [Download EXE File](https://github.com/MaxwellQue/Discord-Tool/archive/refs/heads/main.zip)
2. Run `main.exe`
