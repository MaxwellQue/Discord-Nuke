# Discord-Tool

## Features

- Server Nuke
  - Members
  - Roles
  - Emoji
  - Stickers
  - Channels

- Account Nuke
  - Delete Owned Server
  - Leave all Server
  - Change Owned Server name
  - Modify Account Information

### Setup
1. [Download EXE File](https://github.com/MaxwellQue/Discord-Tool/archive/refs/heads/main.zip)
2. Run `main.exe`
